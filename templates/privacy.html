No personally identifiable information is recorded or transmitted at any time by this app.

If enabled in settings, the app will submit your current position in lat/lng format every X minutes, where X is the interval time also specified in the settings menu (by default this is every hour).

Before submitting these coordinates, they are randomised slightly to give an approximate location.

This is in place so the app is never EXACTLY sure of where you are (just for a little peace of mind!).

This app uses OneSignal to identify each separate device. This is required to keep the map markers updated nicely! Again this ID contains no personally identifiable information, an example ID is below
